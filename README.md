Project Overview

This project focuses on analyzing customer purchasing behavior using Python, SQL, and Power BI to generate actionable business insights. The dashboard helps businesses understand customer demographics, revenue contribution, subscription trends, category-wise sales, and customer review behavior.

The project demonstrates the complete data analytics workflow, including data cleaning, transformation, SQL analysis, and interactive dashboard visualization
